JavaScript предоставляет множество встроенных функций, которые доступны изначально без необходимости их объявления или импорта дополнительных библиотек. Ниже приведены некоторые из наиболее часто используемых встроенных функций в JavaScript:

1. `console.log()` - Используется для вывода сообщений или значений в консоль разработчика.

2. `alert()` - Выводит модальное диалоговое окно с указанным сообщением для пользователя.

3. `prompt()` - Выводит модальное диалоговое окно, запрашивая ввод данных от пользователя.

4. `confirm()` - Выводит модальное диалоговое окно с вопросом и кнопками "ОК" и "Отмена". Возвращает `true`, если пользователь нажал "ОК", и `false`, если пользователь нажал "Отмена".

5. `parseInt()` - Преобразует строку в целое число.

6. `parseFloat()` - Преобразует строку в число с плавающей запятой.

7. `String()` - Преобразует значение в строку.

8. `Number()` - Преобразует значение в число.

9. `Array()` - Создает новый массив.

10. `Object()` - Создает новый объект.

11. `Date()` - Создает новый объект даты и времени.

12. `setTimeout()` - Запускает выполнение функции или кода через указанное время.

13. `setInterval()` - Запускает выполнение функции или кода через указанный интервал времени.

14. `clearTimeout()` - Очищает таймер, созданный с помощью `setTimeout()`.

15. `clearInterval()` - Очищает интервал, созданный с помощью `setInterval()`.

Это только некоторые из множества встроенных функций в JavaScript. Они предоставляют удобные средства для решения различных задач в разработке веб-приложений и работы с данными.

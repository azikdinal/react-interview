Для добавления типов аргументов и возвращаемого значения в функцию в TypeScript, можно использовать аннотацию типа. Вот несколько примеров:

1. Аннотация типа для аргументов функции:
```typescript
function greet(name: string, age: number) {
  console.log(`Hello, ${name}! You are ${age} years old.`);
}
```
В этом примере функция `greet` принимает два аргумента: `name` с типом `string` и `age` с типом `number`.

2. Аннотация типа для возвращаемого значения функции:
```typescript
function add(a: number, b: number): number {
  return a + b;
}
```
В этом примере функция `add` принимает два аргумента типа `number` и возвращает значение типа `number`.

3. Аннотация типа для колбэк функций:
```typescript
function processResult(callback: (result: string) => void) {
  // Логика обработки результата
  const result = "Processed";
  callback(result);
}
```
В этом примере функция `processResult` принимает колбэк функцию с аргументом `result` типа `string` и без возвращаемого значения (`void`).

4. Типизация функции в виде типа:
```typescript
type MathOperation = (a: number, b: number) => number;

const add: MathOperation = (a, b) => a + b;
const subtract: MathOperation = (a, b) => a - b;
```
В этом примере мы определяем тип `MathOperation` как функцию, принимающую два аргумента типа `number` и возвращающую значение типа `number`. Затем мы присваиваем переменным `add` и `subtract` функции с таким типом.

Добавление аннотаций типов для аргументов и возвращаемого значения функции помогает TypeScript проводить статическую проверку типов, обнаруживать ошибки на этапе компиляции и предоставлять подсказки и автодополнение кода в интегрированных средах разработки (IDE).

Обратите внимание, что TypeScript также поддерживает инференцию типов, которая позволяет выводить типы аргументов и возвращаемого значения на основе контекста использования функции. Если типы могут быть выведены автоматически, нет необходимости явно аннотировать их.
